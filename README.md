Для того чтобы скачать и заустить проект нужно выполнить следущее:

1. запустить git  с консоли в нужной директории
   (для этого в нужной папке зажимаем Shift+правая кнопка мышки, 
    нужно выбоать консольный режим)

2. выполняем git clone https://github.com/jtahun/greetTask.git

3. после загрузки заходим в директорию greetTask

4. запускаем cmd.exe (зажимаем Shift+правая кнопка мышки)

5. выполняем команду: mvn package

6. выполняем команду: cd target

7. выполняем команду move ViaAjax.war "$CATALINA_BASE/webapps"
    ("$CATALINA_BASE" - директория где установлен сервер Tomcat
      если в имени папки есть пробелы необходимо второй параметр
      команды move поместить в кавычки пример: move ViaAjax.war "C:\Tomcat 8.5\webapps")

8. запускаем браузер, вводим localhost/ViaAjax/ajax.html
   в случае если Tomcat настроен на порт 8080, 
   вводим localhost:8080/ViaAjax/ajax.html
